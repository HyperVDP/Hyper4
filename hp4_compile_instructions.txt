Instructions for converting .p4 to .hp4:

Number each action in the .p4.  Each action's number serves
as the meta_primitive_state.action_ID, filled in via
match::t1_extracted_exact and match::set_program_state.  Starting
point for this process is each stages::stageX control function.

Handling subtypes: user may define a mapping for each subtype for
a supported primitive.  It is advised to adhere to a standard
map, however.  Here is one for the subtypes in the mod_actions action_profile:

modify_field subtypes:
  meta_stdmeta_ingressport	0
  meta_stdmeta_packetlength	1
  meta_stdmeta_egressspec	2
  meta_stdmeta_egressport	3
  meta_stdmeta_egressinst	4
  meta_stdmeta_insttype		5
  stdmeta_egressspec_meta	6
  meta_const			7
  stdmeta_egressspec_const	8
